About Us

Open source can provide fast, stable, and easy to use embedded solutions.

We continue to look for creative ways to make the embedded solutions better so our users can build innovative products.

About NutsBoard

NutsBoard solutions are based on embedded technologies. Our goal is to build connected platforms for a wide variety of industrial applications to the users.

Our team of experienced engineers offer full open source Software Development Kit and hardware boards. In addition the System-on-Module approach can easily work with a carrier board design. The development time will be greatly reduced by using embedded platform, network or other IoT applications.

Providing an ecofriendly, powerful extended life cycle boards is our mission. If you would like to learn more about how Embedded Linux and Open Source can enhance your productivity, reduce the cost and pollution, please contact us. Together, we can make the world a better place.

What can we bring to you?

Our releases will include hardware, software OS runtime image, SDK source tar ball, and IoT framework API (2nd stage), full open source so any users can do the applications by using our solutions.
